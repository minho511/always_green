## 모듈
****
- 파이썬 코드를 논리적으로 그룹화한 파일.
- `.py` 확장자를 가진 파이썬 파일 하나가 하나의 모듈.
- 모듈은 변수, 함수, 클래스 등을 포함할 수 있으며, 다른 파이썬 코드에서 가져와 사용할 수 있다.

## 패키지
****
- 관련된 모듈을 그룹화한 디렉토리.
- 패키지는 모듈을 계층적으로 구성하여 더 큰 프로그램을 체계적으로 관리할 수 있게함.
- 각 디렉토리는 `__init__.py` 파일을 포함하여 패키지로 인식됨.

### 패키지 생성과 사용

1. 패키지 디렉토리 구조
```
mypackage/
	__init__.py
	module1.py
	module2.py
```
2. 각 모듈파일 작성
```python
# module1.py
def func1():
	return "This is function 1"
# module2.py
def func2():
	return "This is function 2"
```
3. 패키지 초기화 파일 작성
`__init__.py` 파일은 패키지를 초기화하는데 사용됨. 이 파일을 통해 패키지에서 어떤 모듈을 기본적으로 가져올지 정의할 수 있음.
```python

from .module1 import func1
from .module2 import func2
```
4. 패키지 사용
```python
import mypackage

print(mypackage.func1()) # This is function 1
print(mypackage.func2()) # This is function 2

```

### 하위 패키지

패키지는 하위 패키지를 가질 수 있음.
```
mypackage/
	__init__.py
	subpackage/
		__init__.py
		submodule.py
```
