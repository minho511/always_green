- 위치 인자 가변 매개변수
```python
def greet(*names):
	for name in names:
		print(f"Hello, {name}!")

greet("Alice", "Bob", "Charlie")
```
- 키워드 인자 가변 매개변수
```python
def greet(**info):
	for key, value in info.items():
		print(f"{key}: {value}")

greet(name="Alice", age=30)
```
